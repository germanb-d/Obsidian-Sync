# Flashcards
#Mazo

Usando el contenido de {[[Nota Name]]}, genera entre 5 y 10 flashcards importantes para estudiar.

Reglas:
- No escribas explicaciones ni texto fuera de las flashcards.
- Cada tarjeta debe contener una sola idea clara (tarjetas atómicas).
- Mantén las respuestas cortas.
- En las tarjetas bidireccionales usa exclusivamente el símbolo ↔.
- Nunca uses | ni otros separadores para las tarjetas bidireccionales.
- Usa únicamente los siguientes formatos compatibles.

Formatos:

Single-Line Basic
Pregunta → Respuesta

Single-Line Bidirectional
Concepto 1 ↔ Concepto 2

Multi-line Basic
Pregunta
?
Respuesta

Multi-line Bidirectional
Concepto 1
??
Concepto 2

Usa tarjetas bidireccionales solo cuando ambos lados puedan funcionar como pregunta.
Prioriza definiciones, conceptos clave, comparaciones y listas cortas.
