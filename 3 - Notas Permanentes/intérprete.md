Una alternativa a la compilación de un programa escrito en un lenguaje de alto nivel, es usar un intérprete. Un intérprete es un programa que, traduce y ejecuta un programa, instrucción por instrucción. A diferencia del [[Compilador|compilador]], que traduce el programa como un todo y genera un archivo ejecutable que puede ejecutarse tantas veces como se desee.

Para ejecutar un programa, el intérprete ejecuta un loop que consiste en leer las instrucciones del programa de a una a la vez, decidir que es necesario hacer para ejecutarla y luego realizar los comandos necesarios del lenguaje de máquina específico.
