El **código binario** o **lenguaje de alto nivel** consiste de instrucciones muy simples que la CPU de la computadora ejecuta directamente. Cada tipo de procesador tiene su propio lenguaje de máquina. Cada código binario es específico para cada plataforma.

El encargado de traducir el lenguaje de programación de alto nivel al lenguaje máquina es el [[Compilador|Compilador]].
