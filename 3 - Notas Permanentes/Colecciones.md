# Definición

Una **colección** (también conocida como contenedor) es un objeto que contiene un grupo de objetos tratados como una sola unidad. Es decir, permite almacenar, recuperar y manipular múltiples elementos como si fueran un conjunto cohesivo.

## Características Principales:

- **Almacena múltiples objetos**: Puede contener cualquier tipo de objetos
- **Tratamiento unificado**: Todos los elementos se manejan bajo un mismo concepto
- **Operaciones comunes**: Permite agregar, quitar, encontrar, recuperar e iterar sobre los elementos

## Operaciones Básicas Sobre Colecciones:

1. **Agregar** objetos a la colección
2. **Quitar** objetos de la colección
3. **Encontrar** un objeto o grupo de objetos específicos
4. **Recuperar** un objeto (sin eliminarlo de la colección)
5. **Iterar** a través de todos los elementos

## Concepto De "Conocimiento múltiple":

Las colecciones son útiles cuando necesitamos manejar no solo un objeto, sino varios objetos que comparten cierta naturaleza o características comunes. Por ejemplo, una lista de estudiantes, un conjunto de productos, o un mapa de empleados con sus IDs.
