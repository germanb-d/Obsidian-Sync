# Sustantivos (Nouns)

## Componentes De Hardware

1.  **Computer case** ← *Gabinete*
    -   Ejemplo: *I opened the computer case to clean the dust.*
        → Abrí el gabinete de la computadora para limpiar el polvo.
2.  **Motherboard** ← *Placa madre*
    -   Ejemplo: *The motherboard connects all the components together.*
        → La placa madre conecta todos los componentes.
3.  **Hard drive** ← *Disco duro*
    -   Ejemplo: *The hard drive stores all your files and software.*
        → El disco duro guarda todos tus archivos y programas.
4.  **Optical drives** ← *Unidades ópticas*
    -   Ejemplo: *Modern computers rarely use optical drives.*
        → Las computadoras modernas rara vez usan unidades ópticas.
5.  **Processor** ← *Procesador*
    -   Ejemplo: *The processor executes instructions very quickly.*
        → El procesador ejecuta instrucciones muy rápido.
6.  **Silicon chip** ← *Chip de silicio*
    -   Ejemplo: *A silicon chip is inside the CPU.*
        → Un chip de silicio está dentro de la CPU.
7.  **Heat sink** ← *Disipador de calor*
    -   Ejemplo: *The heat sink absorbs the heat from the CPU.*
        → El disipador de calor absorbe el calor de la CPU.
8.  **Power supply unit** ← *Fuente de alimentación*
    -   Ejemplo: *The power supply unit converts power from the wall outlet.*
        → La fuente de alimentación convierte la energía del tomacorriente.

## Conceptos Y Elementos Generales

1.  **Ports** ← *Puertos*
    -   Ejemplo: *You can connect a printer to the USB port.*
        → Podés conectar una impresora al puerto USB.
2.  **Short-term memory** ← *Memoria de corto plazo*
    -   Ejemplo: *RAM is the computer’s short-term memory.*
        → La RAM es la memoria de corto plazo de la computadora.
3.  **The wall outlet** ← *Tomacorriente*
    -   Ejemplo: *Always unplug the computer from the wall outlet first.*
        → Siempre desenchufá la computadora del tomacorriente primero.
4.  **Grounded metal object** ← *Objeto metálico con conexión a tierra*
    -   Ejemplo: *Touch a grounded metal object before opening your PC.*
        → Tocá un objeto metálico conectado a tierra antes de abrir tu PC.
5.  **Head noun** ← *Sustantivo principal*
    -   Ejemplo: En *“the central processing unit”*, el **head noun** es *unit*.
6.  **Buttons** ← *Botones (los presionás)*
    -   Ejemplo: *Press the power button to turn on the computer.*
        → Presioná el botón de encendido para prender la computadora.
7.  **Dials** ← *Perillas (las girás)*
    -   Ejemplo: *Old radios have dials to tune the frequency.*
        → Las radios antiguas tienen perillas para ajustar la frecuencia.

8.  **Strongly typed** ← *Fuertemente tipado*
    -   Ejemplo: *[[Java]] is a **strongly typed** language, so you must declare the variable type.*
        → [[Java]] es un lenguaje fuertemente tipado, por lo que debes declarar el tipo de variable.
9.  **Library support** ← *Soporte de librerías*
    -   Ejemplo: *Python is popular for data science because of its extensive **library support**.*
        → Python es popular para la ciencia de datos debido a su amplio soporte de librerías.
10.  **Systems programming** ← *Programación de sistemas*
    -   Ejemplo: *C++ is often used for **systems programming** due to its performance.*
        → C++ se usa a menudo para la programación de sistemas debido a su rendimiento.
11.  **Garbage collection** ← *Recolección de basura*
    -   Ejemplo: *Automatic **garbage collection** helps manage memory in Java.*
        → La recolección automática de basura ayuda a gestionar la memoria en Java.
12.  **Object-oriented programming** ← *Programación orientada a objetos*
    -   Ejemplo: ***Object-oriented programming** uses concepts like classes and objects.*
        → La programación orientada a objetos usa conceptos como clases y objetos.
13.  **Low-level implementation language** ← *Lenguaje de implementación de bajo nivel*
    -   Ejemplo: *C is considered a **low-level implementation language**.*
        → C es considerado un lenguaje de implementación de bajo nivel.
14.  **Supports dynamic typing** ← *Soporta tipado dinámico*
    -   Ejemplo: *Python **supports dynamic typing**, so a variable can change its type.*
        → Python soporta tipado dinámico, por lo que una variable puede cambiar su tipo.
15. **Interpreted language** ← *Lenguaje interpretado*
    -   Ejemplo: *Python is an interpreted language, which means its code is executed line by line.*
        → Python es un lenguaje interpretado, lo que significa que su código se ejecuta línea por línea.

# Verbos Y Frases Verbales (Verbs and Verbal Phrases)

## Verbos Simples

1.  **Maintain** ← *Mantener / conservar*
    -   Ejemplo: *The OS maintains security of the system.*
        → El sistema operativo mantiene la seguridad del sistema.
2.  **Control** ← *Controlar*
    -   Ejemplo: *The OS controls input and output devices.*
        → El sistema operativo controla los dispositivos de entrada y salida.
3.  **Allow** ← *Permitir*
    -   Ejemplo: *The OS allows users to communicate with the computer.*
        → El sistema operativo permite que los usuarios se comuniquen con la computadora.
4.  **Run** ← *Ejecutar / correr*
    -   Ejemplo: *You need to run the installer file to set up the software.* → Tenés que ejecutar el [[Archivo]] de instalación para configurar el programa.
5.  **Save** ← *Guardar*
    -   Ejemplo: *Don't forget to save your work before you close the program.* → No te olvides de guardar tu trabajo antes de cerrar el programa.
6.  **Store** ← *Almacenar / guardar*
    -   Ejemplo: *Flash drives can store a lot of data.* → Las memorias USB pueden almacenar muchos datos.
7.  **Run** ← *Ejecutar / Funcionar* (Nota: Ya estaba en tu lista, pero es un verbo con muchos usos. Acá va otro contexto).
    -   Ejemplo: *This code won't **run** on the old version of the software.*
        → Este código no se ejecutará en la versión antigua del programa.

## Phrasal Verbs (Verbos Frasales)

1.  **Start up** ← *Arrancar / iniciar*
    -   Ejemplo: *The computer starts up in less than 10 seconds.*
        → La computadora arranca en menos de 10 segundos.
2.  **Carry out** ← *Realizar / ejecutar*
    -   Ejemplo: *The CPU carries out millions of instructions per second.*
        → La CPU ejecuta millones de instrucciones por segundo.
3.  **Deal with** ← *Ocuparse de / encargarse de*
    -   Ejemplo: *The OS deals with application errors.*
        → El sistema operativo se encarga de los errores de las aplicaciones.
4.  **Type in** ← *Tipear / ingresar con teclado*
    -   Ejemplo: *Users type in commands in a CLI.*
        → Los usuarios tipean comandos en una CLI.
5.  **Look after** ← *Encargarse de / administrar*
    -   Ejemplo: *The window manager looks after interactions between windows.*
        → El administrador de ventanas se encarga de la interacción entre ventanas.
6.  **Switch on/off** ← *Encender / apagar*
    -   Ejemplo: *You switch on the computer with the power button.*
        → Encendés la computadora con el botón de encendido.
7.  **Pick up** ← *Detectar / captar*
    -   Ejemplo 1: *The sensor picks up the user’s gesture.* → El sensor detecta el gesto del usuario.
    -   Ejemplo 2: *The microphone picks up every little sound.* → El micrófono capta cada pequeño sonido.
8.  **Work on** ← *Trabajar en*
    -   Ejemplo: *I need to work on my presentation for tomorrow.* → Necesito trabajar en mi presentación para mañana.
9.  **Take up** ← *Ocupar*
    -   Ejemplo: *Videos can take up a lot of disk space.*
        → Los videos pueden ocupar mucho espacio en el disco.
10. **Check out** ← *Revisar / Mirar*
    -   Ejemplo: *Check out these new files I uploaded.*
        → Mirá estos nuevos archivos que subí.
11. **Depend on** ← *Depender de*
    -   Ejemplo: *The speed depends on your internet connection.*
        → La velocidad depende de tu conexión a Internet.
12.  **Figure out** ← *Descifrar / Entender / Resolver*
    -   Ejemplo: *I need to **figure out** why the program is crashing.*
        → Necesito descifrar por qué se está colgando el programa.
13.  **Pop up** ← *Aparecer (de repente)*
    -   Ejemplo: *A warning message **popped up** on the screen.*
        → Un mensaje de advertencia apareció en la pantalla.

## Frases Verbales (Verbal Phrases)

1.  **Improve performance** ← *Mejorar el rendimiento*
    -   Ejemplo: *Adding more RAM can improve performance.*
        → Agregar más RAM puede mejorar el rendimiento.
2.  **Boost the performance** ← *Potenciar / aumentar el rendimiento*
    -   Ejemplo: *An SSD can boost the performance of your computer.*
        → Un SSD puede potenciar el rendimiento de tu computadora.
3.  **Make space** ← *Hacer espacio*
    -   Ejemplo: *I deleted old files to make space for new ones.*
        → Borré archivos viejos para hacer espacio para nuevos.

# Adjetivos (Adjectives)

1.  **Sluggish** ← *Lento/a*
-   Ejemplo: *The system feels sluggish when there is not enough RAM.*
    → El sistema se siente lento cuando no hay suficiente RAM.
1.  **Built-in** ← *Integrado*
    -   Ejemplo: *Many laptops have built-in cameras.*
        → Muchas laptops tienen cámaras integradas.
2.  **Solar-powered** ← *Con energía solar / Alimentado por energía solar*
    -   Ejemplo: *The **solar-powered** calculator works anywhere there is light.*
        → La calculadora con energía solar funciona en cualquier lugar donde haya luz.

# **Frases Y Conectores (Phrases and Connectors)**

1.  **For whatever reason** ← *Por la razón que sea*
    -   Ejemplo: ***For whatever reason**, the application fails to start on this computer.*
        → Por la razón que sea, la aplicación no inicia en esta computadora.
2.  **Intersect with** ← *Intersectarse con / Cruzarse con*
    -   Ejemplo: *My responsibilities **intersect with** the IT department.*
        → Mis responsabilidades se cruzan con las del departamento de TI.
3.  **Comes into play** ← *Entra en juego*
    -   Ejemplo: *This is where your programming knowledge **comes into play**.*
        → Aquí es donde entra en juego tu conocimiento de programación.
4.  **Instead of** ← *En lugar de / En vez de*
    -   Ejemplo: *Use an SSD **instead of** an HDD for better performance.*
        → Usá un SSD en lugar de un HDD para un mejor rendimiento.
5.  **Improve accuracy** ← *Mejorar la precisión*
    -   Ejemplo: *The new algorithm helps to **improve accuracy**.*
        → El nuevo algoritmo ayuda a mejorar la precisión.
6.  **It's a matter of** ← *Es cuestión de*
    -   Ejemplo: ***It's a matter of** personal preference which OS you choose.*
        → Es cuestión de preferencia personal qué sistema operativo elijas.
7.  **Lag a bit behind** ← *Quedarse un poco atrás / Ir con retraso*
    -   Ejemplo: *This model **lags a bit behind** the latest version in terms of features.*
        → Este modelo se queda un poco atrás de la última versión en cuanto a características.
8.  **In a straight line** ← *En línea recta*
    -   Ejemplo: *The robot is programmed to move **in a straight line**.*
        → El robot está programado para moverse en línea recta.
9.  **One major caveat is...** ← *Una gran salvedad / advertencia es...*
    -   Ejemplo: ***One major caveat is** the high cost of the software license.*
        → Una gran salvedad es el alto costo de la licencia del software.
10. **Is only feasible with** ← *Solo es factible con*
    -   Ejemplo: *Real-time video processing **is only feasible with** powerful hardware.*
        → El procesamiento de video en tiempo real solo es factible con hardware potente.
11. **In a nutshell** ← *En pocas palabras / En resumen*
    -   Ejemplo: ***In a nutshell**, the update fixes security issues.*
        → En pocas palabras, la actualización soluciona problemas de seguridad.
12. **Since** ← *Dado que / Ya que* (cuando explica una razón)
    -   Ejemplo: ***Since** the file is corrupted, we cannot open it.*
        → Dado que el [[Archivo]] está corrupto, no podemos abrirlo.
