# JDBC

 JDBC (Java Database Connection) es una API de Java que permite a las aplicaciones Java interactuar con bases de datos relacionales.

Une el paradigma relacional de SQL con el paradigma OO de programación.

- Clase = Tabla
- Instancia = Tupla (Fila)
- Atributo = Columna

## O/R Mapper (ORM)

**Mapeador de Objetos Relacional**

## DAOs

**Data Access Object**

## Drivers

Aplicacion java - Driver JDBC - Base de Datos

El driver cambia dependiendo la Base de Datos.
