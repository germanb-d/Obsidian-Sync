¡Excelente pregunta! Es una de las más comunes en los parciales. Acá te doy la **fórmula exacta** que usan los textos de tus parciales.

---

## Fórmula para responder "What is this text about?"

### Estructura básica

> **The text is about + [tema principal] + [contexto / alcance]**

O también (más común en tus parciales):

> **This text compares/describes/explains/discusses + [qué hace el texto] + [con qué elementos]**

---

## Las 3 variantes según el tipo de texto

| Tipo de texto | Fórmula | Ejemplo (de tus parciales) |
|:---|:---|:---|
| **Comparación** | *The text compares X and Y.* | *"This text compares **Wi-Fi and Bluetooth**."* |
| **Descripción** | *The text describes X.* | *"This text describes **the main differences between Java and Python**."* |
| **Explicación** | *The text explains how X works / why X happens.* | *"This text explains **how Netflix uses matching to pick thumbnails**."* |

---

## Verbos útiles para empezar la respuesta

| Verbo | Cuándo usarlo | Ejemplo |
|:---|:---|:---|
| **compares** | el texto contrasta dos o más cosas | *The text compares Wi-Fi and Bluetooth.* |
| **describes** | el texto explica características de algo | *The text describes the key features of Java.* |
| **explains** | el texto detalla un proceso o causa | *The text explains how reinforcement learning works.* |
| **discusses** | el texto presenta diferentes aspectos de un tema | *The text discusses the problem of algorithmic bias.* |
| **presents** | el texto muestra información sobre algo | *The text presents the main differences between two programming languages.* |
| **analyzes** | el texto examina en profundidad | *The text analyzes the impact of AI on daily life.* |

## Estructura paso a paso

### Paso 1: Identificá el tipo de texto
- ¿Compara dos cosas? → **compares**
- ¿Describe características? → **describes**
- ¿Explica un proceso? → **explains**
- ¿Presenta información general? → **presents / discusses**

### Paso 2: Identificá el tema principal
- ¿De qué habla? (ej: reinforcement learning, algorithmic bias, Netflix)

### Paso 3: Armá la oración

> **This text + [verbo] + [tema] + (opcional: contexto)**

## ⚠️ Errores comunes que tenés que evitar

| ❌ Incorrecto | ✅ Correcto | ¿Por qué? |
|:---|:---|:---|
| *The text is about of Wi-Fi* | *The text is about Wi-Fi* | "About" ya incluye la idea, no hace falta "of" |
| *The text talk about Java* | *The text **talks** about Java* | El verbo necesita concordancia (3ª persona -s) |
| *The text explain how works Netflix* | *The text **explains** how Netflix **works*** | El orden de los verbos es sujeto + verbo |
| *Is about the differences* | *The text is about the differences* | La oración necesita sujeto ("The text") |
