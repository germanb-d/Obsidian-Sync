# 🟦 RESUMEN PARA APROBAR EL PARCIAL - INGLÉS INTEGRAL 2 🟦

---

## 1. TIEMPOS VERBALES (Los que más aparecen)

| Tiempo | Cuándo usarlo | Estructura | Señal |
|:---|:---|:---|:---|
| **Present Simple** | Hechos, verdades generales, rutinas | Sujeto + verbo (con **-s** en he/she/it) | *always, never, usually, every day* |
| **Present Continuous** | Ahora mismo, tendencias, arreglos futuros | **am/is/are + verbo-ing** | *now, at the moment, currently* |
| **Past Simple** | Acción terminada en un momento específico | Verbo en pasado (-ed o irregular) | *yesterday, last week, in 1970* |
| **Present Perfect** | Experiencia, cambio, o acción que empezó en el pasado y sigue | **have/has + participio pasado** | *for, since, already, yet, ever, never* |

### ⚠️ ERROR GRAVE QUE TE SALVA:
- **TO BE** (am/is/are/was/were) NUNCA usa *did* en negativo
- ❌ *did not be* → ✅ *was not / were not*

---

## 2. THAN (Para comparar)

| Regla | Ejemplo |
|:---|:---|
| Adjetivo corto + **-er + than** | *Wi-Fi is **faster than** Bluetooth* |
| **more/less +** adjetivo largo + **than** | *Python is **more intuitive than** Java* |
| Irregulares: **better / worse than** | *This algorithm is **better than** the other* |

### ⚠️ NUNCA:
- ❌ *more faster than* (doble comparativo)
- ❌ *better that* (es **than**, no *that*)

---

## 3. PREGUNTAS NEGATIVAS (Tabla simple)

| Tiempo | Estructura corta | Ejemplo |
|:---|:---|:---|
| **Present Simple** | **Don't / Doesn't** + sujeto + verbo? | *Doesn't he work?* |
| **Past Simple** | **Didn't** + sujeto + verbo base? | *Didn't you go?* |
| **Past Perfect** | **Hadn't** + sujeto + participio? | *Hadn't they finished?* |

### ⚠️ TO BE en Past Simple:
- *Wasn't he there?* / *Weren't they ready?*

---

## 4. RELATIVE CLAUSES (La diferencia clave)

| | **DEFINING** (especificativa) | **NON-DEFINING** (explicativa) |
|:---|:---|:---|
| **Función** | Identifica CUÁL | Da info EXTRA |
| **Comas** | ❌ Sin comas | ✅ CON comas |
| **Se usa "that"** | ✅ Sí | ❌ NO |
| **Se omite pronombre** | ✅ Sí (si es objeto) | ❌ NO |

### Pronombres:
- **who** → personas
- **which** → cosas
- **that** → personas o cosas (solo DEFINING)
- **where** → lugares
- **whose** → posesión

### Ejemplo mínimo:
- *The algorithm **that** works fast is useful.* (DEFINING)
- *The algorithm, **which** works fast, is useful.* (NON-DEFINING)

---

## 5. NOUN PHRASES (Encontrar el HEAD NOUN)

### Fórmula:
> **[Determinante] + [Adjetivos] + HEAD NOUN + [complemento]**

### Ejemplos de parciales:

| Frase | HEAD NOUN | Traducción |
|:---|:---|:---|
| *the number of bytes they can hold* | **number** | el número de bytes que pueden contener |
| *a person's short-term memory* | **memory** | la memoria de corto plazo de una persona |
| *a device that uses NAND flash* | **device** | un dispositivo que usa NAND flash |

### Cómo encontrar el HEAD NOUN:
1. Preguntate: ¿De qué se habla?
2. Sacá adjetivos y complementos
3. Lo que queda es el núcleo

## 6. LINKING EXPRESSIONS (Lo que preguntan en el parcial)

| Conector | Relación | Traducción |
|:---|:---|:---|
| *however* | **contrast** | sin embargo |
| *therefore* / *so* / *as a result* | **consequence** | por lo tanto |
| *because* / *since* | **cause** | porque / dado que |
| *for example* / *for instance* | **example** | por ejemplo |
| *first* / *then* / *next* | **temporal/sequence** | primero / luego |
| *furthermore* / *moreover* | **addition** | además |

### Pregunta típica:
> *"What relationship does 'however' express?"*
> **Respuesta:** *"It expresses contrast."*

## 7. DEFINICIONES (Fórmula de 3 partes)

### Estructura:
> **TÉRMINO + GRUPO AL QUE PERTENECE + THAT + CARACTERÍSTICA DISTINTIVA**

### Ejemplo:
> *"Reinforcement learning is a machine learning training method **that** rewards desired behaviors and punishes undesired ones."*

| Parte | Ejemplo |
|:---|:---|
| Término | *Reinforcement learning* |
| Grupo | *is a machine learning training method* |
| Distintivo (con THAT) | *that rewards desired behaviors...* |

### ⚠️ NUNCA usar "is when" o "is where"

---

## 8. WHAT IS THIS TEXT ABOUT? (Fórmula)

> **This text + [verbo] + [tema principal]**

### Verbos según el tipo de texto:

| Texto | Verbo |
|:---|:---|
| Compara dos cosas | **compares** |
| Describe características | **describes** |
| Explica un proceso | **explains** |
| Presenta información general | **presents / discusses** |

### Ejemplos:
- *"This text **compares** Wi-Fi and Bluetooth."*
- *"This text **describes** the key features of Java and Python."*
- *"This text **explains** how reinforcement learning works."*

## 10. PASADO SIMPLE NEGATIVO (La trampa)

| ✅ Correcto | ❌ Incorrecto |
|:---|:---|
| *I **did not work*** | *I did not **worked*** |
| *He **didn't go*** | *He didn't **went*** |
| *She **was not*** | *She **did not be*** |

### Regla de oro:
- Verbos normales → **did not + verbo BASE**
- TO BE (was/were) → **was not / were not** (sin *did*)

---

## 11. TRUCO PARA RESPONDER TRUE/FALSE EN EL PARCIAL

1. **Buscá la parte del texto** que habla de ese tema
2. **Comprará** con la oración del ejercicio
3. Si dice lo mismo → **TRUE**
4. Si dice algo diferente o lo opuesto → **FALSE**
5. **Siempre** señalá las líneas que justifican tu respuesta

---

## 12. TRUCO PARA TRADUCIR NOUN PHRASES

1. Encontrá el **HEAD NOUN**
2. Traducí primero el head noun
3. Después traducí los adjetivos (van antes)
4. Después traducí los complementos (van después, con "de" o "que")

### Ejemplo:
> *"the risks of interference with other devices"*
1. Head noun = **risks**
2. "los riesgos"
3. "de interferencia"
4. "con otros dispositivos"
