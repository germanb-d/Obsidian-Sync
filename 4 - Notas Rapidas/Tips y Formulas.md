
## 1) Para responder: “WHAT IS THIS TEXT ABOUT?”

No pongas una oración larga ni detalles secundarios.
### Fórmulas útiles:

- **This text is about + topic**
- **The text is about + topic**
- **The main topic of the text is + topic**
- **This text mainly deals with + topic**
### Qué tiene que tener tu respuesta:

- tema general
- pocas palabras
- sin ejemplo innecesario
- sin copiar todo el texto
### Ejemplo de formato:

- **This text is about computer components and their functions.**
- **The main topic of the text is the use of hardware in a computer system.**
    
---
## 2) Fórmula de definición de 3 partes

Esta es clave. La estructura más segura es:

**TERM + IS A/AN + GROUP/CATEGORY + THAT + DISTINCTIVE CHARACTERISTIC**

### Ejemplos:

- **A processor is a component that executes instructions.**
- **A motherboard is a main circuit board that connects all the parts of a computer.**
- **A driver is a program that allows the operating system to communicate with hardware.**
### Qué está pasando ahí:

- **TERM**: la palabra que definís.
- **GROUP/CATEGORY**: a qué clase pertenece.
- **THAT + characteristic**: qué hace o qué lo distingue.

Si no usás esa estructura, la definición queda floja.

---
## 3) Cómo responder definiciones sin dar vueltas

Si te preguntan “What is X?”, respondé así:
- **X is a/an + category + that + function**
- **X is a/an + thing/device/program + that + verb**
    
---
## 4) Para preguntas de comprensión

### Si te piden idea principal:
- respondé global, no detallado.
### Si te piden información específica:
- buscá nombres, fechas, causas, consecuencias, comparaciones.
### Si te piden verdadero/falso:
No alcanza con decir **True** o **False**. Hay que justificar con una parte del texto.
### Si te piden completar:
No inventes. Mirá:
- tiempo verbal
- preposición
- singular/plural
- sujeto concordante

---
## 5) Cómo estudiar para carpeta abierta

Tu carpeta no tiene que ser “bonita”. Tiene que ser **buscable**:
- títulos grandes
- subtítulos claros
- fórmulas al inicio
- ejemplos mínimos pero potentes
- palabras clave resaltadas
