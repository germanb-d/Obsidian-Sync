# NOTA 1 — VOCABULARIO ÚTIL PARA TEXTOS DE COMPUTACIÓN / HARDWARE

Acá no alcanza con traducir palabras sueltas. Tenés que reconocer **familias de vocabulario** y cómo se usan en definiciones y textos explicativos.

## 1) Partes y componentes

- **CPU / processor**: procesador, el “cerebro” de la computadora.
    
- **GPU / graphics card**: placa o tarjeta gráfica.
    
- **motherboard**: placa madre.
    
- **RAM**: memoria temporal.
    
- **storage**: almacenamiento.
    
- **hard drive / SSD**: disco rígido / unidad de estado sólido.
    
- **power supply**: fuente de alimentación.
    
- **cooling system**: sistema de refrigeración.
    
- **fan**: ventilador.
    
- **slot**: ranura.
    
- **cable**: cable.
    
- **port**: puerto.
    
- **driver**: controlador.
    
- **device**: dispositivo.
    
- **hardware**: parte física.
    
- **software**: programa o sistema.
    
- **peripheral**: periférico.
    

## 2) Verbos que aparecen mucho

- **to install**: instalar.
    
- **to connect**: conectar.
    
- **to disconnect**: desconectar.
    
- **to upgrade**: mejorar / actualizar un componente.
    
- **to replace**: reemplazar.
    
- **to repair**: reparar.
    
- **to run**: funcionar / ejecutar.
    
- **to support**: soportar / ser compatible con.
    
- **to detect**: detectar.
    
- **to damage**: dañar.
    
- **to overheat**: sobrecalentarse.
    
- **to fail**: fallar.
    
- **to configure**: configurar.
    
- **to process**: procesar.
